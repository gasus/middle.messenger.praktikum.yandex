## Описание

Проект чата для 1го модуля курса

## Технологии

- Vite
- TypeScript
- Express
- LESS

## Установка

- `npm i` — установка
- `npm run dev` — запуск версии для разработчика без сборки
- `npm run start` — запуск версии для разработчика со сборкой
- `npm run build` — сборка бандла
